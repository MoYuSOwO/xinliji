# 新礼记写作

## 宗旨

将现代素材、网间议论、谐语笑谈与社会小事，写成可收入《新礼记》的轻文言小品。文气近于《世说新语》、志人笔记、志怪短札与古雅评论区，不取经学论文、奏疏檄文、八股讲章之气。

成品宜有古意、机锋、温情或讽趣；重点是“以小事见人情，以戏语见义理”。

## 项目定制

统一使用项目名《新礼记》。“周礼新议”只作卷首、序文或内在论题，不另作项目总名。

固定卷目与 `volumeId` 如下：

- `0` 序：周礼新议
- `1` 卷一 · 修身卷：情绪、自控、成长、学习、身体、心性
- `2` 卷二 · 家门卷：亲子、家庭、婚恋、代际、亲情、人伦
- `3` 卷三 · 仕学卷：学校、考试、导师、职场、老板、同事
- `4` 卷四 · 货殖卷：钱财、消费、购物、订阅、贫富、交易
- `5` 卷五 · 百工卷：技术、AI、编程、工程、器物、车、工具
- `6` 卷六 · 世风卷：网梗、舆论、争议、社会怪象、评论区

不要新增 `tags`、`summary`、`subtitle`、`source`、`sourceUrl`。本项目不用这些字段。

## 输出形态

聊天中默认给出：

```text
推荐题名：……
可归入：《新礼记·某卷·某篇》

正文：

【题名】

……

论曰：……
```

若要直接写入项目文章，生成 `articles/` 下的 `.md` 文件：

```md
---
title: "题名"
volumeId: 6
date: "2026-06-30"
author: "MoYuSOwO"
---

正文……

论曰：……
```

篇名自然时，可用中文文件名，如 `articles/东坡训稚.md`。中文文件名可用；避免空格和复杂标点。目录默认按卷 id、日期倒序、题名排序，不写手工序号。日期默认用当前日期；用户另有指定则从其指定。作者默认用 `author: "MoYuSOwO"`；用户另有指定则从其指定。

添加或改写文章后，运行 `npm run build` 或 `npm run check`，确保 `generated/articles.json` 与正文一致。

## 写作流程

1. 提炼事件：找出“谁做了什么”、何处可笑、何处可感、何处有误会或反转。
2. 取味道句：找出最有记忆点的一句话、一个动作或一个旁人反应。
3. 定卷与题：按六卷择其一；题名优先二至五字，像古籍篇名，不像公众号标题。
4. 改写正文：开头交代人物与事件，中段写众议、误会或对话，末段以一句话、一个动作或一处反转收住。
5. 写“论曰”：一段即可；先点破人情，再收束义理。宜轻、准、短，不要说教。
6. 自查成稿：须有清楚小事、可记之句、转折余味、轻文言文气，并以 `论曰：` 收束。

用户未给篇名时，拟三至八个题名并推荐一个。用户未指定卷目时，自行选择最贴切的 `volumeId`，并简短说明归入何卷。

## 文体准则

采用“世说新语型轻文言”：

- 短句为主，偶用长句。
- 叙事要简洁，有人物、有动作、有一句关键对话。
- 议论含蓄，不堆典故。
- 古雅但可读，不艰深，不端着。
- 可保留轻微网感，但须转写为古雅机趣。
- 留余味，不必把意思全讲尽。

避免：

- 大量堆叠“夫、盖、然则、故曰”。
- 过度引经据典。
- 把小事写成奏疏、檄文、论文或道德讲章。
- 逐句把现代口语硬译成假古文。
- 让“论曰”变成训人。

普通篇目篇末固定用 `论曰：`。除非用户要求总序、序文或特别庄重篇，不用 `赞曰`、`故曰`。

## 常用结构

误会反转型：

```text
某人有某行，众见而笑之。
或讥曰：“……”
其人答曰：“……”
众闻之，或惭，或默，或叹。

论曰：……
```

众议型：

```text
某事出，众论纷然。
或曰：“……”
或曰：“……”
或解之曰：“……”
于是议稍定。

论曰：……
```

志人型：

```text
某人素如何。
一日遇某事，众皆如何。
某人独曰：“……”
闻者异之。

论曰：……
```

礼学公案型：

```text
或问：“某事合于礼乎？”
一曰守礼。
一曰论功。
一曰辨形。
一曰通变。
众议乃定。

论曰：……
```

## 词语转写

现代词可作如下转写：

- 网友：网间人、众人、观者、好事者
- 评论区：众议、网间、群中
- 视频：影像、网录
- 博主：录者、其人、某子
- 梗：谐语、戏语、笑谈
- 破防：愤然、失色、不能平
- 反转：众乃悟、闻者始异之
- 吵架：争辨、相诘、哗然
- 抽象：怪诞、诙奇、荒唐而有味
- 情绪价值：慰人心、解人郁、使人稍安
- 道德绑架：挟义责人、以名分困人

常用句式：

- 近岁，网间有……
- 或见而笑曰……
- 其人徐答曰……
- 众闻之，始默然。
- 有识者曰……
- 此非……也，乃……也。
- 不亦……乎？
- 岂独……哉？
- 观者初笑之，既而叹之。

## 成稿自查

交付前确认：

- 是否围绕一个清楚的小事件？
- 题名是否像古籍篇名？
- `volumeId` 是否符合六卷体系？
- 是否有一句可记之语？
- 文气是否轻文言，而非僵硬古文？
- 篇末是否以 `论曰：` 起段？
- 是否没有新增 `tags`、`summary`、`subtitle`、`source`、`sourceUrl`？
